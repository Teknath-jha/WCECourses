Python for Everyone


Department of CSE has launched value added Programme on Python for Everybody under Center for Continuing Education (CCE). Emerging trends in technical and professional education call for value addition i.e. students must acquire something more than what is provided in the curriculum. The college offers numerous of value added courses which are designed to bridge the gap between the academia and industry. By doing this VAP course participant excel in Python basics


Programme Objectives

  1. To provide in depth knowledge of basic concepts of python programming.
  2. To demonstrate working of Python standard library such as Numpy and Pandas.
  3. To explore Python's object oriented features.
  4. To develop the skill of providing solution for any real life problem using Python.

Course Content

  1. Introduction to python
  2. Working with Data types
  3. Exception handling and file handling using python
  4. Object oriented programming using python
  5. Introduction to NumPy
  6. Introduction to Pandas
